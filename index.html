📦 Množice

📌 KAJ JE MNOŽICA?

Množica je zbirka različnih stvari. Tem stvarem rečemo ELEMENTI.

Primer: A = {1, 2, 3} je množica s tremi elementi: 1, 2 in 3.

🔤 POMEN SIMBOLOV

∈ = "je element" → 2 ∈ {1,2,3} (drži)

∉ = "ni element" → 5 ∉ {1,2,3} (drži)

⊂ = "je podmnožica" → {2} ⊂ {1,2,3} (drži)

∅ = prazna množica – nima nobenega elementa. Je podmnožica vsake množice.

🔗 OPERACIJE Z MNOŽICAMI

A ∪ B (unija) = vse kar je v A ali v B (ali obeh)

A ∩ B (presek) = samo tisti elementi, ki so v A in v B (skupni)

A \ B (razlika) = elementi v A, ki niso v B

Ac (komplement) = vse razen A (znotraj univerzalne množice U)

Primer: A = {1,2,3}, B = {3,4,5}
A ∪ B = {1,2,3,4,5}
A ∩ B = {3}
A \ B = {1,2}

📜 DE MORGANOVA ZAKONA

(A ∪ B)c = Ac ∩ Bc

(A ∩ B)c = Ac ∪ Bc

🔄 KARTEZIČNI PRODUKT

A × B = {(a,b) : a ∈ A, b ∈ B} – vsi možni urejeni pari

|A × B| = |A| · |B|

📐 INTERVALI (na številski premici)

Interval je neprekinjen kos številske premice.

[2,5] = zaprt (vključno 2 in 5)

(2,5) = odprt (brez 2 in 5)

(2,5] = polodprt (brez 2, z 5)

[2,5) = polodprt (z 2, brez 5)

(2,∞) = vsa števila večja od 2

(-∞,2] = vsa števila manjša ali enaka 2

Unija intervalov (∪) = združitev dveh intervalov.
Primer: (-∞, -3) ∪ (7, ∞)

⚠️ NEENAČBE (INEQUALITIES)

< = manjše    > = večje    ≤ = manjše ali enako    ≥ = večje ali enako

Linearna neenačba: 2x + 3 > 7 → 2x > 4 → x > 2 → rešitev: (2, ∞)

Kvadratna neenačba: x² - 4 < 0 → x = -2 ali x = 2 → parabola odprta navzgor → rešitev: (-2, 2)

Neenačba z absolutno vrednostjo: |x - 2| < 5 → -5 < x-2 < 5 → -3 < x < 7 → rešitev: (-3, 7)

Pomembno: Rešitev neenačbe je VEDNO MNOŽICA (interval ali unija intervalov)!

🧠 Izjave in logika

📌 KAJ JE IZJAVA?

Izjava je trditev, ki je lahko PRAVILNA (P) ali NAPAČNA (N).

Primer: "2 + 2 = 4" je PRAVILNA izjava.

Primer: "2 + 2 = 5" je NAPAČNA izjava.

🔗 LOGIČNI VEZNIKI

P ∧ Q (P in Q) – resnično le, če sta obe resnični

P ∨ Q (P ali Q) – resnično, če je vsaj ena resnična

P ⇒ Q (če P potem Q) – napačno le, če je P resničen in Q napačen

P ⇔ Q (P natanko tedaj ko Q) – resnično, ko imata P in Q isto vrednost

¬P (negacija) – nasprotna vrednost

⚠️ POMEMBNO: P ⇒ Q je resničen tudi, ko je P napačen!

🔍 KVANTIFIKATORJA

∀ (za vsak) – velja za vse elemente

∃ (obstaja) – obstaja vsaj en element

📜 NEGACIJE KVANTIFIKATORJEV

¬(∀x P(x)) = ∃x ¬P(x)

¬(∃x P(x)) = ∀x ¬P(x)

¬(P ⇒ Q) = P ∧ ¬Q
